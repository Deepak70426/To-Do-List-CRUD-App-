# To-Do-List-CRUD-App-

Designed a To-Do List using JavaScript ( DOM Manipulation),

HTML, CSS (Bootstrap)

to perform CRUD operations (create, read, update, delete).

// WORKING
Input the task and add it to to-do list by clicking Add Button.

To rename the task ,use EDIT button to update the task.

After the task is done, or to delete the task, use REMOVE button.
